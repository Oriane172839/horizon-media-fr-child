# Horizon — Média indépendant

Horizon est un média indépendant en ligne traitant d’actualité, de société et de sujets de fond avec une approche pédagogique et accessible.
**Live website:** https://horizonmedia.fr


## Objectif du projet

Développer une plateforme média moderne, rapide et lisible, capable d’accueillir une publication régulière d’articles avec une forte exigence éditoriale.

Le site doit :
- Offrir une lecture confortable sur tous supports  
- Mettre en avant les contenus éditoriaux  
- Être optimisé pour le référencement naturel  
- Permettre une gestion éditoriale fluide  

## Stack technique

- WordPress
- Thème basé sur Blogvi / Bloglo (structure légère et adaptable)
- Personnalisations via CSS et ajustements de mise en page
- Optimisations SEO on-page
- Optimisations de performance (structure légère, médias optimisés)

## Travail réalisé

- Adaptation du design pour un rendu plus éditorial et moderne  
- Optimisation des pages d’articles pour la lisibilité  
- Mise en place d’une hiérarchie claire des titres et contenus  
- Personnalisation des éléments visuels (catégories, badges, blocs de mise en avant)  
- Amélioration de l’expérience utilisateur sur mobile  

## Approche technique

Le projet a été structuré avec une logique de maintenabilité :
- Personnalisations réalisées sans modification du cœur du thème  
- Séparation claire entre design, contenu et logique fonctionnelle  
- Code organisé pour faciliter les évolutions futures  

## Statut

Projet en production avec mises à jour éditoriales régulières.  
Ce repository présente la structure et les personnalisations mises en place.
